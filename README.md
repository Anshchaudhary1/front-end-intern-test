# Front-End Intern Test - EZ Labs (React, Vite compatible)

This is a lightweight React app (Vite-compatible) created for an intern assignment.
It uses plain CSS (no Tailwind) and includes:
- Navbar, Hero, Features placeholder
- Contact form integrated with https://vernanbackend.ezlab.in/api/contact-us/
- Basic front-end validation (required fields + email format)
- Success / error messages

How to run locally (only if you have Node & npm):
1. npm install
2. npm run dev
3. Open the shown URL

If your laptop doesn't support Node, upload this project to an online editor:
- StackBlitz (https://stackblitz.com) -> Import from ZIP
- Vercel / Netlify for deployment
